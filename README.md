# Коллоквиум 1
## Задания
### 1. Факториалы
**Вход:** `n` (натуральное число)  
**Выход:** вектор первых `n` факториалов  
**Пример:** `n=5` → `[1, 2, 6, 24, 120]`

### 2. Удаление дубликатов  
**Вход:** вектор чисел  
**Выход:** вектор без дубликатов (сохраняя порядок)  
**Пример:** `[1,2,2,3,4,4,5]` → `[1,2,3,4,5]`

### 3. Разворот списка (рекурсия)
**Вход:** головной узел односвязного списка  
**Выход:** развёрнутый список  
**Пример:** `1→2→3→null` → `3→2→1→null`

## Требования
- Корректность работы
- Эффективность алгоритмов
- Читаемость кода
- Код максимально близкий к промышленному
- Юнит-тесты
- Обработка специальных и исключительных ситуаций
